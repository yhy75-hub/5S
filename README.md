# 事務所 5Sチェックシート

iPad・スマホ・PCどこからでも使える5S活動チェックシートアプリです。

---

## ファイル構成

```
5Sアプリ/
├── Code.gs          ← GASサーバーサイドロジック
├── index.html       ← フロントエンド（iPad/スマホ対応）
├── appsscript.json  ← GASマニフェスト
└── README.md        ← このファイル
```

---

## 初回セットアップ手順

### 1. Googleスプレッドシートを作成

1. Googleドライブで「**新規 → Googleスプレッドシート**」を作成
2. 名前をつけて保存（例：`事務所5S管理`）

### 2. Apps Scriptを開く

1. スプレッドシートのメニュー「**拡張機能 → Apps Script**」を開く
2. 左側のファイル一覧に `コード.gs` が表示される

### 3. コードを貼り付ける

**Code.gs：**
1. `コード.gs` を開いてデフォルトの内容をすべて削除
2. `Code.gs` の内容をコピーして貼り付け
3. 保存（Ctrl+S）

**index.html：**
1. 左のファイル一覧の「＋」→「**HTMLファイル**」→ 名前を `index` にして追加
2. `index.html` の内容をコピーして貼り付け
3. 保存（Ctrl+S）

### 4. PINコードを設定（初回のみ）

1. スクリプトエディタの関数選択欄から `setAdminPin` を選択
2. 実行（▶ボタン）
3. ※ デフォルトのPINは `5s2026`。変更する場合は `Code.gs` の関数内の値を書き換えてから実行

### 5. Webアプリとしてデプロイ

1. 右上「**デプロイ → 新しいデプロイ**」をクリック
2. 種類の選択 → 「**ウェブアプリ**」
3. 設定：
   - 説明：`事務所5Sチェックシート`
   - 次のユーザーとして実行：`自分`
   - アクセスできるユーザー：`全員`（社外アクセス不要なら `組織内全員`）
4. 「**デプロイ**」をクリック
5. 表示された **URL** をコピー → メンバーに共有

## 日常の使い方

### メンバー（通常月）
1. URLをブックマーク or QRコードで開く
2. 自分の名前をタップ
3. ○△×で各項目を評価
4. 「提出する」ボタン → **提出後は変更不可**

### 管理者・5Sリーダー（通常月）
1. URLを開いて名前を選択
2. PINコードを入力
3. メンバータブを切り替えながら評価入力（第三者評価月のみ）
4. 共通エリア（棚・壁柱・防火防災）を評価

### 第三者評価月（3/6/9/12月）
- メンバーの評価画面は自動でブロック
- 管理者・リーダーのみがすべての評価を担当

### 結果確認（管理者のみ）
- ログイン後「📊 結果一覧」タブ
- ○=緑 △=黄 ×=赤 で色分け表示
- CSVダウンロード可能（Excelで開ける）

## カスタマイズ

`Code.gs` の上部 `CFG` を変更：

```js
const CFG = {
  members: ['平林','森崎', ...],    // ← メンバー名を変更
  privileged: ['伊藤A','本城A', ...], // ← 管理者・リーダー名を変更
  qMonths: [3, 6, 9, 12],           // ← 第三者評価月を変更
};
```

変更後は「**デプロイ → デプロイを管理 → 新しいバージョン**」で再デプロイが必要。

---

## PINコードの変更方法

1. Apps Scriptエディタを開く
2. `setAdminPin` 関数内の文字列を新しいPINに変更
3. 関数を選択して実行
4. ※ コード上のPINはダミー。実際の値はGoogleのPropertiesServiceに安全に保存される

---

## データ確認

スプレッドシートに月ごとのシート（例：`2026-06`）が自動作成され、
提出済みデータが記録されます。スプレッドシート上でも集計・グラフ作成が可能です。
