# 事務所 5Sチェックシート

iPad・スマホ・PCどこからでも使える5S活動チェックシートアプリ。

**GitHub Pages** でフロントエンドを配信し、**Google Apps Script** がAPIサーバー、**Googleスプレッドシート** がデータベースとして機能します。

---

## アーキテクチャ

```
GitHub Pages (index.html)
    ↕  fetch() API呼び出し
Google Apps Script (Code.gs) ← REST API
    ↕
Googleスプレッドシート ← データ保存
```

---

## ファイル構成

```
5Sアプリ/
├── index.html       ← フロントエンド（GitHub Pagesで配信）
├── Code.gs          ← GAS APIサーバー
├── appsscript.json  ← GASマニフェスト
└── README.md
```

## セットアップ手順

### Step 1: Google Apps Script の準備

1. [Googleスプレッドシート](https://sheets.google.com)を新規作成（名前：`事務所5S管理`など）
2. メニュー「**拡張機能 → Apps Script**」を開く
3. `コード.gs` の内容を `Code.gs` の内容に全部置き換えて保存
4. 左の「＋」→「HTMLファイル」→ 名前を `index` として追加（今回は不要なので空でOK）

### Step 2: PINコードを設定

1. スクリプトエディタの関数選択で `setAdminPin` を選んで実行
2. 初回はGoogleアカウントの承認を求められるので許可する
3. ※ PINを変えたい場合は `Code.gs` 内の `'5s2026'` を変更してから実行

### Step 3: GASをWebアプリとしてデプロイ

1. 右上「**デプロイ → 新しいデプロイ**」
2. 種類：**ウェブアプリ**
3. 設定：
   - 次のユーザーとして実行：`自分`
   - アクセス：`全員`
4. 「デプロイ」→ 表示された **URL をコピー**（後で使う）

### Step 4: GitHub Pages を有効化

1. GitHubリポジトリの「**Settings → Pages**」を開く
2. Branch: `main` / Folder: `/ (root)` → **Save**
3. 公開URL（`https://yhy75-hub.github.io/5S/`）が発行される

### Step 5: アプリを開いてGAS URLを登録

1. GitHub PagesのURLをブラウザで開く
2. 初回起動時にGAS URLの入力画面が表示される
3. Step 3 でコピーしたURLを貼り付けて「設定して開始」

## 日常の使い方

### メンバー（通常月）
- URLを開いて自分の名前をタップ
- ○(10点)／△(5点)／×(0点) で評価
- 「提出する」→ **提出後は変更不可**

### 管理者・5Sリーダー（いつでも）
- 名前を選んでPINでログイン
- メンバータブを切り替えて評価入力
- 共通エリア（棚・壁柱・防火防災）も同画面で評価
- 「📊 結果一覧」で色分け表示・CSVダウンロード

### 第三者評価月（3月・6月・9月・12月）
- メンバーログインは自動でブロック
- 管理者・リーダーのみが全メンバーを評価

## カスタマイズ

`Code.gs` 上部の `CFG` を変更してGASを再デプロイ：

```js
const CFG = {
  members:    ['平林','森崎', ...],   // ← メンバー名
  privileged: ['伊藤A','本城A', ...], // ← 管理者・リーダー名
  qMonths:    [3, 6, 9, 12],          // ← 第三者評価月
};
```

> 変更後は「デプロイ → デプロイを管理 → 新しいバージョン」で再デプロイが必要

---

## データ確認

スプレッドシートに月ごとのシート（`2026-06` など）が自動作成され、提出済みデータが記録されます。スプレッドシート上でそのまま集計・グラフ作成も可能です。
